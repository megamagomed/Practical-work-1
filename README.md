## –§–∞–π–ª `HEAD`

–§–∞–π–ª `HEAD` (–∞–Ω–≥–ª. ¬´–≥–æ–ª–æ–≤–∞¬ª, ¬´–≥–æ–ª–æ–≤–Ω–æ–π¬ª) ‚Äî –æ–¥–∏–Ω –∏–∑ —Å–ª—É–∂–µ–±–Ω—ã—Ö —Ñ–∞–π–ª–æ–≤ –ø–∞–ø–∫–∏ `.git`. –û–Ω —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ –∫–æ–º–º–∏—Ç, –∫–æ—Ç–æ—Ä—ã–π —Å–¥–µ–ª–∞–Ω –ø–æ—Å–ª–µ–¥–Ω–∏–º (—Ç–æ –µ—Å—Ç—å –Ω–∞ —Å–∞–º—ã–π –Ω–æ–≤—ã–π).

–í —ç—Ç–æ–º –º–æ–∂–Ω–æ —É–±–µ–¥–∏—Ç—å—Å—è —Å –ø–æ–º–æ—â—å—é —Ç–µ—Ä–º–∏–Ω–∞–ª–∞. –ü–µ—Ä–µ–π–¥–∏—Ç–µ –≤ –ø–∞–ø–∫—É `.git` –∫–æ–º–∞–Ω–¥–æ–π `cd`. –ü–æ—Å–º–æ—Ç—Ä–∏—Ç–µ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ —Ñ–∞–π–ª–∞ `HEAD` –∫–æ–º–∞–Ω–¥–æ–π `cat`.

```bash

$ cd .git/

$ cat HEAD # –∫–æ–º–∞–Ω–¥–∞ cat –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ —Ñ–∞–π–ª–∞
```

–í–Ω—É—Ç—Ä–∏ `HEAD` ‚Äî —Å—Å—ã–ª–∫–∞ –Ω–∞ —Å–ª—É–∂–µ–±–Ω—ã–π —Ñ–∞–π–ª: `refs/heads/master` (–∏–ª–∏ `refs/heads/main` –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –Ω–∞–∑–≤–∞–Ω–∏—è –≤–µ—Ç–∫–∏). –ï—Å–ª–∏ –∑–∞–≥–ª—è–Ω—É—Ç—å –≤ —ç—Ç–æ—Ç —Ñ–∞–π–ª, –º–æ–∂–Ω–æ —É–≤–∏–¥–µ—Ç—å —Ö–µ—à –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ –∫–æ–º–º–∏—Ç–∞.

```bash

    –î–æ–±–∞–≤–∏—Ç—å –∞–º–±–∏—Ü–∏–π –≤ —Å–ø–∏—Å–æ–∫ –¥–µ–ª
```

–ö–æ–≥–¥–∞ –≤—ã –¥–µ–ª–∞–µ—Ç–µ –∫–æ–º–º–∏—Ç, Git –æ–±–Ω–æ–≤–ª—è–µ—Ç `refs/heads/master` ‚Äî –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç –≤ –Ω–µ–≥–æ —Ö–µ—à –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ –∫–æ–º–º–∏—Ç–∞. –ü–æ–ª—É—á–∞–µ—Ç—Å—è, —á—Ç–æ `HEAD` —Ç–æ–∂–µ –æ–±–Ω–æ–≤–ª—è–µ—Ç—Å—è, —Ç–∞–∫ –∫–∞–∫ —Å—Å—ã–ª–∞–µ—Ç—Å—è –Ω–∞ `refs/heads/master`.

–ü—Ä–∏ —Ä–∞–±–æ—Ç–µ —Å Git —É–∫–∞–∑–∞—Ç–µ–ª—å `HEAD` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–æ–≤–æ–ª—å–Ω–æ —á–∞—Å—Ç–æ. –ú—ã —É–∂–µ —É–ø–æ–º–∏–Ω–∞–ª–∏, —á—Ç–æ –º–Ω–æ–≥–∏–µ –∫–æ–º–∞–Ω–¥—ã Git –ø—Ä–∏–Ω–∏–º–∞—é—Ç –≤ –∫–∞—á–µ—Å—Ç–≤–µ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞ —Ö–µ—à –∫–æ–º–º–∏—Ç–∞. –ï—Å–ª–∏ –Ω—É–∂–Ω–æ –ø–µ—Ä–µ–¥–∞—Ç—å –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç, —Ç–æ –≤–º–µ—Å—Ç–æ –µ–≥–æ —Ö–µ—à–∞ –º–æ–∂–Ω–æ –ø—Ä–æ—Å—Ç–æ –Ω–∞–ø–∏—Å–∞—Ç—å —Å–ª–æ–≤–æ `HEAD` ‚Äî Git –ø–æ–π–º—ë—Ç, —á—Ç–æ –≤—ã –∏–º–µ–ª–∏ –≤ –≤–∏–¥—É –ø–æ—Å–ª–µ–¥–Ω–∏–π –∫–æ–º–º–∏—Ç.

# –°—Ç–∞—Ç—É—Å—ã —Ñ–∞–π–ª–æ–≤ –≤ Git

–î–æ –ø–æ—è–≤–ª–µ–Ω–∏—è Git —Å–∏—Å—Ç–µ–º—ã –∫–æ–Ω—Ç—Ä–æ–ª—è –≤–µ—Ä—Å–∏–π –≤—ã–¥–µ–ª—è–ª–∏ —Ç–æ–ª—å–∫–æ –¥–≤–∞ —Å—Ç–∞—Ç—É—Å–∞ —É —Ñ–∞–π–ª–æ–≤: ¬´—É–∂–µ –∑–∞–∫–æ–º–º–∏—á–µ–Ω¬ª –∏ ¬´–µ—â—ë –Ω–µ –∑–∞–∫–æ–º–º–∏—á–µ–Ω¬ª. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ Subversion (—Å–∞–º–æ–π –ø–æ–ø—É–ª—è—Ä–Ω–æ–π VCS –¥–æ —ç–ø–æ—Ö–∏ Git) –Ω–µ –Ω—É–∂–Ω–æ –±—ã–ª–æ –≤—ã–ø–æ–ª–Ω—è—Ç—å –∫–æ–º–∞–Ω–¥—É ‚Äî –∞–Ω–∞–ª–æ–≥ `git add`, –∞ –º–æ–∂–Ω–æ –±—ã–ª–æ –ø—Ä–æ—Å—Ç–æ —Å–¥–µ–ª–∞—Ç—å –∫–æ–º–º–∏—Ç (`svn commit`). –≠—Ç–∞ –∫–æ–º–∞–Ω–¥–∞ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –¥–æ–±–∞–≤–ª—è–ª–∞ –≤ –∫–æ–º–º–∏—Ç –≤—Å–µ –Ω–æ–≤—ã–µ –∏ –∏–∑–º–µ–Ω—ë–Ω–Ω—ã–µ —Ñ–∞–π–ª—ã.

## –°—Ç–∞—Ç—É—Å—ã `untracked`/`tracked`, `staged` –∏ `modified`

* staged (–∞–Ω–≥–ª. ¬´–ø–æ–¥–≥–æ—Ç–æ–≤–ª–µ–Ω–Ω—ã–π¬ª)
